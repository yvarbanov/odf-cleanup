# odf-cleanup
Removes leftover odf images based on an identifier

## Requires
- python3.6
- ceph/odf tools 
- odf conf
- odf keyring with sufficient privileges

## How to
```console
git clone https://github.com/yvarbanov/odf-cleanup.git odf-cleanup
python3 odf-cleanup.py
```
